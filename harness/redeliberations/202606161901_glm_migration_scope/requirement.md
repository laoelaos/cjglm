# 需求：确定 GLM 1.0.3 迁移的基础类型范围

## 背景
项目 `/references/glm-1.0.3` 是 OpenGL Mathematics (GLM) 库 1.0.3 版本，一个 C++ 的图形学数学库。现在计划用另一门语言重新实现该库。

## 任务
分析 GLM 1.0.3 项目，确定：
1. 哪些类型是最基础的（被最多其他类型/函数依赖），应该被优先迁移
2. 首轮迁移的范围应该是多大比较合适（包含哪些类型及其关系）
3. 为什么选这些类型作为首轮迁移目标

## 要求
- **只确定范围，不做具体设计和实现**
- 分析项目中的类型依赖关系，给出明确的范围建议
- 给出分层的迁移建议（哪些必须第一批，哪些可以后续迁移）
- 考虑类型之间的依赖关系，确保首轮迁移的类型可以独立编译/工作
