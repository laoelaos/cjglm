### P1. [严重] S3 分析中存在事实错误——round-trip 测试实际可捕获 quatCast 因子 2 bug

**位置：** §S3 分析段，第 68-69 行

**问题描述：** 报告声称「由于 quatCast 因子 2 bug 是整体缩放，round-trip 测试无法捕获（缩放因子在往返中自抵消）」。此断言在数学上不成立。对于非身份四元数的 round-trip（如 `testMat3CastNonIdentityQuat`、`testQuatCastNonIdentityMat3RoundTrip`、`testQuatCastNonIdentityMat4RoundTrip`），quatCast 返回的错误四元数分量（非最大分量约 2x）进入 `mat3Cast` 后，矩阵元素涉及分量的二次型（x²、xy 等），导致结果矩阵与原矩阵显著不同，round-trip 断言 `@Expect(m == m2, true)` 将 FAIL。仅在身份四元数 round-trip 测试中（非最大分量为 0，缩放后仍为 0）才无法捕获该 bug。

**影响传播：**
- 优先级表第 3 项（S3）理由「若 S1 已修复，本项优先级降为 Low」基于错误前提

- S3 的「测试方法论改进」子建议仍然有效，但核心断言需要修正

**建议：** 修正 S3 分析：区分身份与非身份 round-trip 测试的不同捕获能力；删除「缩放因子在往返中自抵消」的错误表述；在优先级表中移除 S1 修复后 S3 自动降级的假设。
