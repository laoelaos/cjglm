## 需要解决的问题（仅 RETRY 时存在）

- **所在位置**：§3.1 common.cj 职责，第 248 行
- **严重程度**：一般
- **改进建议**：将公式修正为 `exponent = floor(log2(abs(x))) + 1`

- **所在位置**：§3.3 gtc/ulp.cj 实现路径，第 684 行
- **严重程度**：一般
- **改进建议**：修正为基于 x 自身位模式的操作，如 `T.fromBits(x.toBits() + 1u) - x`
