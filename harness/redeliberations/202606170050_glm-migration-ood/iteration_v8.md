- **严重程度**：严重

- **问题描述**：设计未利用 `public import` 定义公共 API 面，下游消费者导入路径不明
- **所在位置**：§2（模块划分，第 34-51 行）、§3.8（第 244-271 行）
- **严重程度**：严重
- **改进建议**：在 `package glm` 中新增公共 API 入口文件（如 `lib.cj`/`mod.cj`），使用 `public import` 重导出 `glm.detail` 中的 Vec 类型和 Qualifier 类型；在 §2 模块划分中明确公共 API 面设计；在 §12 验收标准中增加公共 API 导入可用性验收项

- **问题描述**：缺少与 roadmap 首轮范围的可追溯性映射，编码阶段无法独立验证范围完整性
- **所在位置**：§1（概述，第 9 行）、§8（迁移文件清单与依赖拓扑，第 689-715 行）
- **严重程度**：严重

- **严重程度**：一般
